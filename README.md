# np-profiling

**Microscopy-based natural product (NP) profiling using Cell Painting and machine learning.**

This pipeline takes high-content Cell Painting image data from compound libraries, normalizes it, performs image quality control, and trains Random Forest classifiers to predict compound mechanism of action — at the drug target level and across ATC pharmacological classification levels 1–4.

Data is stored in a [DuckDB](https://duckdb.org/) embedded database throughout, which allows fast in-process SQL queries without a database server.

## Pipeline Overview

Scripts are run **in numbered order** from the project root. Each script checks whether it has
already run against the current database and skips gracefully — pass `--force` to override.

| Script | Description |
|--------|-------------|
| **01** | Loads training library and NP fraction `.RData` files, concatenates them, runs a 3-pass plate-level normalization (IQM → plate-median sweep → density-peak centering), and writes all feature matrices and metadata to `data/db/cellpainting.duckdb`. |
| **02** | Detects dead and empty wells using TIFF image file size as a cell-density proxy. Normalizes file sizes using bimodal density peaks, applies multi-channel cutoffs (DAPI, TxRed, Cy5), and writes a `deadwells_filesize` quality column to the database. |
| **03** | Loads the cleaned database, removes low-quality plates and redundant features, computes per-sample Mahalanobis distances (phenotypic activity), remaps DrugBank target annotations to curated functional classes, and writes the purged data back to the database. |
| **04** | Constructs annotated training sets at five annotation levels (ATC levels 1–4 and drug target) by matching compounds to DrugBank and ATC hierarchies. Saves training sets to `data/output/full_db_training_scale_dba_remap.Rdata`. |
| **05** | Trains one-vs-one (OvO) Random Forest classifiers at each annotation level using repeated cross-validation. Saves per-level model objects to `results/RF_<level>.Rds`. |

## Running the Pipeline

All scripts support the `--wd` flag to set the working directory. Run from the project root:

```bash
Rscript code/01.compile_libraries_fractions_normalize_duck.R --wd /path/to/np-profiling
Rscript code/02.detectDeadBlurry_filesize_duck.R             --wd /path/to/np-profiling
Rscript code/03.create_cc_comp_fractions_dba_duck.R          --wd /path/to/np-profiling
Rscript code/04.fuzzy_create_full_db_training_duck.R         --wd /path/to/np-profiling
Rscript code/05.fuzzyRF_OvO_roc_duck.R                      --wd /path/to/np-profiling
```

### Supported flags

| Flag | Scripts | Description |
|------|---------|-------------|
| `--wd <path>` | 01–05 | Set working directory |
| `--force` | 01–05 | Re-run even if the script has already been applied to this database |
| `--norm <method>` | 01 | Normalization method: `density` (default) or `iqm` |
| `--hqp <name>` | 01, 02 | Set the analyst name for the provenance tag |
| `--emerg` | 01 | Emergency mode: rebuild DuckDB from an existing `.Rds` fallback file |
| `--method <method>` | 02 | Dead-well detection method (currently only `filesize` is supported) |
| `--annotateonly` | 02 | Skip file-size scanning; only re-run the annotation/export step |
| `--masterlist` | 03 | Export a full metadata TSV to `data/output/masterdata.tsv` |
| `--useremaps` | 04 | Use curated DrugBank target-to-functional-class remaps |
| `--nogpt` | 01 | Use original (non-GPT-boosted) DrugBank annotations for Bioactives2 library |

### Image file scanning (script 02)

Script 02 sources `functions/justGetFileSizes.R` to scan raw TIFF images. Before running,
update the `basedir1` and `basedir2` paths in that file to point to your microscopy storage drives.

|------|-------------|
| `data/db/cellpainting.duckdb` | Primary database — normalized features, metadata, quality flags, and processing log |
| `data/output/cp_compiled_all_fractions_norm.Rds` | Fallback legacy object (mirrors DuckDB content) |

| `data/output/deadwells_all.csv` | Flat CSV of all flagged dead wells |
| `data/output/dead_wells_fractions.xlsx` | Annotated dead-well report for NP fraction plates |
| `data/output/full_db_training_scale_dba_remap.Rdata` | Annotated training sets by ATC level and drug target |
| `data/output/RF_training_annotations.Rds` | Compiled class-level summaries |
| `results/RF_<level>.Rds` | Trained Random Forest models (one per annotation level) |
| `logs/images/*.png` | QC scatter plots at each normalization pass |
